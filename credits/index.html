Asset attribution and review status

Castia Market uses a mix of local item artwork and external avatar rendering. The local asset library is being reviewed file by file so each image can be tied to a documented source and reuse basis.

Current source buckets

Source bucket Current handling
Minecraft Wiki-derived files Reviewed individually; local copies stay in place until their exact source and reuse basis are recorded.
CastiaMC-origin assets Permission should be confirmed in writing or the assets should be replaced over time.
Crafty avatars Loaded from Crafty as external avatar renders rather than bundled local assets.
