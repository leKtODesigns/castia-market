# Draft: CastiaMC Asset Permission Request

Subject: Permission request for CastiaMC item artwork on fan-made price tracker

Hello CastiaMC team,

I run Castia Market, a non-commercial fan-made price tracker for Castia players. The site is meant only as an in-game economy guide so players can better understand auction pricing and avoid being under-sold, over-sold, or scammed. It does not broker real-money trades, sell items, or process payments.

I would like written permission to display CastiaMC-origin item artwork on the site, especially server-specific assets such as gear-set icons, runestones, and unique relic images. The artwork would be used only to visually identify items in the price tracker.

The site already states that it is unofficial and not approved by, endorsed by, or associated with CastiaMC, Mojang, Microsoft, or Crafty. I am also maintaining an asset manifest so the origin and permission status of third-party artwork can be documented clearly.

If you are willing to grant permission, please confirm whether Castia Market may:

1. Display CastiaMC-origin item artwork on the public site.
2. Store local copies of those images in the public repository for website delivery.
3. Resize or optimize the files for web use without altering the underlying artwork.

If there are attribution requirements, limits, or assets you do not want used, please include those as well and I will follow them.

Thank you.
